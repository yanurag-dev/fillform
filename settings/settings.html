📖 कैसे उपयोग करें / How to Use

  1. सेटिंग्स में Gemini API key सेव करें / Save your Gemini API key above
  2. जिस फॉर्म को भरना है वो वेबसाइट खोलें / Open the form website
  3. Extension icon क्लिक करें या Alt+F दबाएं / Click the extension icon or press Alt+F
  4. दस्तावेज़ की फोटो अपलोड करें (आधार, मार्कशीट, आदि) / Upload document photos
  5. "डेटा निकालें" क्लिक करें / Click "Extract Data"
  6. डेटा जांचें और सही करें / Review and correct the extracted data
  7. "फॉर्म भरें" क्लिक करें / Click "Fill Form"
  8. फॉर्म सबमिट करें / Submit the form
  9. अगले पेज पर दोबारा "फॉर्म भरें" क्लिक करें / For multi-page forms, click "Fill Form" again on each page

🔧 समस्या निवारण / Troubleshooting

  • फ़ील्ड नहीं भरे / Fields not filling — पेज रीफ्रेश करें और दोबारा कोशिश करें / Refresh the page and try again
  • API key error — ऊपर सेटिंग्स में API key जांचें / Check your API key in settings above
  • डेटा गलत निकला / Wrong data extracted — दस्तावेज़ की साफ फोटो लें / Take a clearer photo of the document
  • गलत मैपिंग / Wrong mapping — नीचे उस URL की कैश मैपिंग मिटाएं / Delete that URL's cached mapping below
  • इंटरनेट नहीं / No internet — केवल कैश्ड फॉर्म के लिए काम करेगा / Only cached forms will work offline
